# excel-cli
some useful actions for excel files

```
NAME:
   excel-cli - some useful actions for excel files

USAGE:
   excel-cli [global options] command [command options] [arguments...]

VERSION:
   v1.0.0

AUTHOR:
   Mark Ma <ma_k@ctrip.com>

COMMANDS:
   split, sp  split sheet into multiple files
   unify, un  unify sheets in multiple files into one sheet
   help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --path value, -p value  root path of pending files (default: "/Users/mklyy/Desktop/")
   --help, -h              show help (default: false)
   --version, -v           print the version (default: false)

COPYRIGHT:
   (c) Authored By Mark Ma
```
